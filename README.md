# Desafio Front-end Web Kinvo

## Instruções para rodar o projeto:

1. Clone o projeto na sua máquina.
2. Acesse a pasta do projeto pelo terminal.
3. Instale as dependências utilizando um dos comandos abaixo:
```shell
// utilizando npm
npm install

// utilizando yarn
yarn install
```
4. Inicie o projeto rodando um dos comandos abaixo:
```shell
// utilizando npm
npm start

// utilizando yarn
yarn start
```
5. Uma nova aba será aberta no navegador, caso não ocorra, abra uma aba no navegar e navegue para localhost:3000
